PyAIE is a Python-based programming framework for AMD Versal ACAP AI Engines. It lets users describe algorithm-level kernels in Python and automatically generates AI Engine kernel code, host code, and configuration scripts.

Links: [DAC 2023 Late Breaking Results](https://ieeexplore.ieee.org/abstract/document/10247843), [LATTE 2023 paper](https://capra.cs.cornell.edu/latte23/paper/10.pdf), [video](https://www.youtube.com/watch?v=QrtP7Fri4pI).
